Boardmaker 7 Editor is the world’s leading symbol-based special education solution. This application supports learning and communication in the classroom, therapy room, or home with a trove of customizable material and Picture Communication Symbols® (PCS).

Early Concepts & Design Guidelines

Boardmaker 7 Editor is a continuation of an already existing product, Boardmaker Studio. The vision for the product was to improve features, introduce a modern and cohesive UI, and support cloud storage and syncing.

The design had to work across multiple platforms that not only aligned with our design guidelines but seamlessly integrated into each supported platform. This was crucial when adding elements that could easily have the design switch to their operating system's. An example of this is the file menu. We strategically used the styling from each supported operating system so that the user did not have to learn a new workflow.
