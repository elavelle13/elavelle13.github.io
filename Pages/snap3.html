Software

Role:UI/UX

Duration: April 2021 - July 2021

TD Snap is a flexible software offering professionals and families a choice of AAC solutions. It comes with a set of powerful tools and resources to empower people with speech and language disabilities to communicate.

Biggest Hurdle

When reviewing the workflows in the onboarding experience, there was one area in particular where we struggled to determine how we wanted the page to behave. In the onboarding experience, we require users to determine what page set they would like to use for communication purposes.

The challenge we faced was that there were multiple layers involved when selecting a page set. Tough decisions were made when it came to how much information we needed to provide to users in order for them to feel confident that they selected the right page set. We also wanted to be transparent as possible when showing what page sets were included and what page sets came at an additional cost.

The solution we found was limiting the amount of text on the first page so that the user could explore the list of page sets without being overwhelmed. If they wanted to learn more about the page sets, then they had the choice to select it and view all the options available with a brief description that could be toggled on/off. This allowed us to clean up the UI but provide information if a user needed it.
