Mobile App

Eye Gaze

Role:UI/UX

Duration: March 2016 - November 2020

Boardmaker 7 Student Center is an app that brings personalized visuals and symbol-supported educational materials home. Assigned activities appear automatically in the learner’s account and can use custom settings for eye gaze, switch, mouse and touch.

Early Concepts & Design Guidelines

The vision for this product was to improve the UI so it had a more modern look, address user pain points and improve the user experience, and introduce new features.

The design had to work for a wide range of students with different access methods as well as educators, guardians, and SLPs. This was the driving force for the design and layout of the application since elements on the screen had to be accessible for all access methods. This required integrating design guidelines that were already established in other eye-tracking products. One example of this is button size and spacing. In order for elements to be accessible, there had to be enough space to select with eye-gaze as well as not strain the eye from having to be accurate as possible when trying to select a specific thing.
